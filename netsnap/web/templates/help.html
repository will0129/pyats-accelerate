Netsnap Help & Documentation

How to use the portal effectively.

1. Getting Started

Before you begin, ensure you have a valid inventory file.

Step 1: Download Template - Go to Dashboard and download the CSV template locally.

Step 2: Fill Inventory - Add your devices (Hostname, IP, Role) to the CSV.

Step 3: Upload Inventory - Upload the filled CSV via the "Upload Inventory" button. This will initialize the testbed.

2. Operations

Capture Snapshot: Connects to all devices in the inventory and captures their current state (Interfaces, CPU, Memory). Saved with a timestamp.

Validate State: Captures a fresh snapshot and compares it against a selected Baseline Snapshot. Generates a report of deviations.

3. Role Permissions

  • Read Only: Can view dashboard and run Validations/Captures (but not modify inventory).
  • Power User: Can upload inventory (modify topology) and run all operations.
  • Admin User: Can manage users and has full access.

4. Troubleshooting

If connections fail, ensure the backend server has network access to the devices and `NETWORK_USER`/`NETWORK_PASS` environment variables are set correctly.
